## Mac에 Jekyll 설치하기
터미널에 아래와 같이 입력을 하면 Jekyll 설치가 끝이 납니다.

```

$ xcode-select --install
\curl -sSL https://get.rvm.io | bash -s stable
$ souce ~/.prifle
$ rvm install 2.2.0
$ gem install jekyll
$ gem install bundler
$ gem install github-pages

```
